## 라즈베리파이 버전 및 OS
  - raspberry model 3B+ 
  - 라즈비안 OS full(32bit)

## update & upgrade
  - sudo apt update
  - sudo apt upgrade

## 음성 데이터 송수신
  - sudo apt install pulseaudio-module-bluetooth 
  - pulseaudio -k
  - pulseaudio --start

## 블루투스 패키지 다운
  - sudo apt-get install bluetooth blueman bluez
  - sudo apt-get install python-bluetooth  ->  sudo apt-get install python3-bluetooth
